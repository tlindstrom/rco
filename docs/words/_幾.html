幾

幾 (jǐ) is a very functional word, it can be used to form questions regarding quantity or functions as a few, some or several.

幾 to ask quantity questions

To ask for quantity of something, we can use 幾 (jǐ). Like numerals, we need a classifier to connect between 幾 (jǐ) and its noun. Since it is followed by a classifier, we can always count the noun with its classifier, for example, in three bowls of rice, rice itself is uncountable but we can still quantify rice with the number of bowls. Therefore, 幾 (jǐ) is often translated into how many. It has the following sentence structure:
幾 + classifier + noun

你 有 幾 個 家 ？
nǐ yǒu jǐ ge jiā ？
How many homes you have?
你 穿 了 幾 件 衣服 ？
nǐ chuān le jǐ jiàn yīfú ？
How many clothes have you worn?
你 喝 了 幾 杯 紅酒 了 ？
nǐ hē le jǐ bēi hóngjiǔ le ？
How many glasses of wine did you drink?

幾 as a few, several or some

幾 (jǐ) is used when we want to avoid mentioning an exact number. The number could be from 1-9, but usually refers to a number larger than 2 and smaller than 10. However, just like a few, several or some in English, there is no consensus on this. Same as when 幾 (jǐ) functions as how many, it requires a classifier: 幾 + classifier + noun
Here are some examples:

我 有 幾 個 朋友 十分 喜歡 唱歌 。
wǒ yǒu jǐ ge péngyǒu shífēn xǐhuān chànggē 。
I have a few friends who love to sing.
有 幾 個 人 躺 在 地上 。
yǒu jǐ ge rén tǎng zài dìshàng 。
There are some people lying on the ground.
我 病 了 幾 個 月 了 。
wǒ bìng le jǐ ge yuè le 。
I was sick for several months.

In English, there are expressions like a few hundred or a few thousand. In Chinese, we have similar expressions but we start with ' a few ten', which does not exist in English. The sentence structure of this kind of expression is like this:
幾 + 十/百/千/萬...+ classifier + noun
For example:

我們 有 幾千 個 員工 ， 別 擔心 。
wǒmen yǒu jīqiān ge yuángōng ， bié dānxīn 。
We have a few thousand employees, don't worry.
那裡 有 幾十 個 人 。
nàlǐ yǒu jīshí ge rén 。
There are 'a few ten' people.
我們 今天 賺 了 幾百 塊 錢 。
wǒmen jīntiān zhuàn le jībǎi kuài qián 。
We earned a few hundred dollars today.

As mentioned before, 幾 (jǐ) is used to replace numbers from 1-9. In situations where we want to avoid the units digit of a 2-digit number, we can use 幾 (jǐ) instead. This kind of expression is similar to those expressions like twenty something or thirty something in English. However, there are some differences:
1) The Chinese equivalents for, for example, twenty-something or thirty-something, do not include the number twenty and thirty respectively;
2) there is also an expression of ' ten something' in Chinese;
3) the English expressions are often used to describe age but in Chinese, they can be used for anything.
We put 幾 (jǐ) in a sentence like this:
十, 二十,三十... + 幾 + classifier + noun
For example:

那裡 有 二十幾 個 人 在 叫 。
nàlǐ yǒu èrshíjī ge rén zài jiào 。
There are twenty-something people screaming.
我 只 有 十幾 張 相片 。
wǒ zhǐ yǒu shíjī zhāng xiāngpiàn 。
I only have 'ten-something' photos.

Note that 幾 (jǐ) requires classifiers only when there is an existing classifier for the noun to be described, and this applies too all the meanings of 幾 (jǐ). The following is some examples in which the nouns do not have a classifier:

我 三十幾 歲 。
wǒ sānshíjī suì 。
I am thirty-something.
我 去 了 幾 天 。
wǒ qù le jǐ tiān 。
I was there for several days.
你 幾點 要 走 ？
nǐ jīdiǎn yào zǒu ？
At what time do you need to leave?
