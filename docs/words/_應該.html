應該

HSK Level: 3

應該 (yīnggāi) can be translated into should in English. It is used to talk about social obligations, suggestions and possibility. It bears a similar sentence structure to that of should as well:
subject + 應該 + verb/ verb phrase

應該 to talk about social obligations

你 不 應該 對 別人 這麼 粗魯 ！
nǐ bù yīnggāi duì biérén zhème cūlǔ ！
You shouldn't be so rude to others!
我們 應該 尊重 父母 。
wǒmen yīnggāi zūnzhòng fùmǔ 。
We should respect our parents.
我們 不 應該 把 垃圾 扔 在 街 上 。
wǒmen bù yīnggāi bǎ lājī rēng zài jiē shàng 。
We should not throw rubbish on the street.
所有 家庭 都 應該 節省 能源 。
suǒyǒu jiātíng dōu yīnggāi jiéshěng néngyuán 。
All families should save energy.

應該 to talk about suggestions

We can use 應該 (yīnggāi) when we want to give/ask for suggestions or when we talk about what we think is good to do. For example:

我們 應該 非常 小心 。
wǒmen yīnggāi fēicháng xiǎoxīn 。
We should be very careful.
你 應該 早點兒 起床 。
nǐ yīnggāi zǎodiǎnr qǐchuáng 。
You should wake up earlier.
我 應該 多 做 運動 減肥 嗎 ？
wǒ yīnggāi duō zuò yùndòng jiǎnféi ma ？
Should I exercise more to lose weight?
如果 我們 要 趕上 這 班 火車 ， 現在 就 應該 走 。
rúguǒ wǒmen yào gǎnshàng zhè bān huǒchē ， xiànzài jiù yīnggāi zǒu 。
If we want to catch the train, we should leave now.

應該 to talk about possibility

When we want to express something is likely or expected to happen, we can use 應該(yīnggāi) as well.

爸爸 回來 的 時候 ， 媽媽 應該 在 家 。
bàba huílái de shíhòu ， māma yīnggāi zài jiā 。
Mum should be at home by the time father comes back.
你 應該 會 覺得 這 個 博物館 很 有趣 。
nǐ yīnggāi huì juéde zhè ge bówùguǎn hěn yǒuqù 。
You should find the museum very interesting.
這 應該 煮熟 了 吧 。
zhè yīnggāi zhǔshú le ba 。
It should be well cooked already.
我 聖誕節 應該 在 泰國 。
wǒ shèngdànjié yīnggāi zài tàiguó 。
I should be in Thailand during Christmas.
