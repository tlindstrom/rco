沒

HSK Level: 1

沒 (méi) is used for negation. It is used to negate the verb 有 (yǒu, for 'to have') , to express that something did not happen and something has not been experienced.

to negate 有

We put 沒 (méi) before 有 (yǒu) to express that one does not have something.
沒 + 有 + noun

我 沒 有 錢 。
wǒ méi yǒu qián 。
I don't have money.
他 沒 有 第二 個 手機 。
tā méi yǒu dìèr gè shǒujī 。
He doesn't have a second phone.
這裡 沒 有 鬼 。
zhèlǐ méi yǒu guǐ 。
There aren't any ghosts.

有 (yǒu) is optional, so we can use 沒 (méi) as a short form of 沒有 (méiyǒu). The last example can therefore be written as:

這裡 沒 鬼 。
zhèlǐ méi guǐ 。
There aren't any ghosts.

to negate a past event

沒 (méi) is used before a verb to express that something did not happen.
Subject + 沒 + 有 (optional) + verb

我 沒 殺 他 。
wǒ méi shā tā 。
I didn't kill him.
我們 沒 吃 那 塊 蛋糕 。
wǒmen méi chī nà kuài dàngāo 。
We didn't eat that piece of cake.
媽媽 沒有 把 鑰匙 給 我 。
māma méiyǒu bǎ yàoshi gěi wǒ 。
My mum hasn't given me the key.

Note that if we want to negate stative verbs or habitual events in the past, we use another negator 不 (bù) instead. For example:

我 以前 不 喜歡 紅蘿蔔 ， 但 現在 喜歡 了 。
wǒ yǐqián bù xǐhuān hóngluóbo ， dàn xiànzài xǐhuān le 。
I didn't like carrots before, but now I do.
我 小 時候 不 撒謊 。
wǒ xiǎo shíhòu bù sāhuǎng 。
I did not lie when I was a kid.
幾 十 年 前 ， 人們 不 知道 什麼 是 電腦 。
jǐ shí nián qián ， rénmén bù zhīdào shénme shì diànnǎo 。
Few decades ago, people did not know what computers were.

to express that something has not been experienced

沒 (méi) often goes with the experiential marker 過 (guò) to describe that someone does not have a certain experience. The sentence structure is as the following:
subject + 沒 + 有(optional) + verb + 過

我 沒 試 過 兔 肉 。
wǒ méi shì guò tù ròu 。
I have never tried rabbit meat.
我 沒有 去 過 北京 。
wǒ méiyǒu qù guò běijīng 。
I have never been to Beijing.
我 沒 抽 過 煙 。
wǒ méi chōu guò yān 。
I have never smoked.

As mentioned in the previous section, 不 (bù) is used to negate stative verbs. We should however be aware that we still use 沒 (méi) when talking about experience. For example:

我 從來 沒 愛 過 你 。
wǒ cónglái méi ài guò nǐ 。
I have never loved you.
我 從 沒 想 過 你 。
wǒ cóng méi xiǎng guò nǐ 。
I have never missed you.

If we want to express that someone has not experienced something without using the experiential marker 過 (guò) , we need to specify in the sentence for how long something has not been experienced. We should use the following structure:
subject + period of time + 沒 + verb + 了

她 三 天 沒 吃飯 了 。
tā sān tiān méi chī le 。
She hasn't eaten for three days.
我們 十 年 沒 見 了 。
wǒmen shí nián méi jiàn le 。
We haven't seen each other for 10 years.
我 很 久 沒 游泳 了 。
wǒ hěn jiǔ méi yóuyǒng le 。
I haven't swum for a long time.
