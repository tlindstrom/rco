死(sǐ) is a taboo word in Chinese culture and many try to avoid this word in daily life. For example, it is not unusual that there is no 4th floor in a building. Since ‘four’ in Mandarin is pronounced as sì, which is similar to sǐ.
Having said that, in spoken Chinese, we use the phrase 死了(sǐle) as a degree complement. It is very similar to extremely or very much in English.
Let's see how it is used in our daily language:

As adjective degree complement ##

adjective + 死了
For example:
這條裙子醜死了！
zhè tiáo qúnzǐ chǒu sǐle ！
This dress is so ugly!
那條街臭死了！
nà tiáo jiē chòu sǐle ！
That street stinks so bad!
我累死了！
wǒ léi sǐle ！
I am so tired!
Compared with the word 很 (hěn), 死了 (sǐle) has a stronger intensifying power. Be aware that we cannot use them in the same sentence, no matter how much we want to emphasize something! Furthermore, it should not not be used in polite company, but, rather, only among close friends and family.
死了(sǐle) used to have a negative connotation but recently it has become more common to use it with positive adjectives as well, just like in the following:
她漂亮死了！
tā piāoliàng sǐle ！
She is extremely beautiful!
他的家乾淨死了！
tāde jiā qiánjìng sǐle ！
His house is so clean!

As a degree complement for stative verbs ##

Verb + 死 + object + 了
For example:
我想死你了！
wǒ xiǎng sǐ nǐ le ！
I miss you so much!
我恨死你了！
wǒ hèn sǐ nǐ le ！
I hate you so much!
Sometimes 死(sǐ) can be used as a result complement to describe the result of its verb/adjective. So we really have to pay extra attention to the intended meaning of the sentence!
For example, the following sentence can have two completely different meanings:
他餓死了！
tā è sǐle ！
He starved to death/ he is extremely hungry.
Note that when 死(sǐ) follows an action verb, it functions as a result complement, not a degree complement. Let's look at some examples now:
他把他打死了。
tā bǎ tā dǎ sǐ le 。
He beat him to death.
部分人在示威中被擠死了。
bùfēn rén zài shìwēi zhōng bèi jǐ sǐ le 。
Some people were crushed to death in the demonstration.
